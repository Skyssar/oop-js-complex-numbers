Los números complejos se pueden sumar, restar, multiplicar y dividir usando las mismas ideas que con los radicales y las variables.
Para sumar o restar, combinar términos semejantes.
Para multiplicar se realiza la propiedad distributiva y se multiplican los términos resultantes como monomios.
Para dividir, tratar el cociente como una fracción.
